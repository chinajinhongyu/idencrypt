# idencrypt
encrypt and decrypt numeric ID.

 * 加解密数字格式的ID,例如将 123456 加密为 38654321894938303106 格式.
 * 加密过程中有校验防篡改功能,加密完成的字符串中修改任意一位均不能正常解密
 * 在应用过程中可以防止爬虫按递增规律抓取
 * 实现效果类似这种格式的网址
